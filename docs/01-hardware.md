## 네트워크 요구사항

```
┌─────────────────────────────────────────────┐
│              공유기/스위치                    │
├─────────────────────────────────────────────┤
│                                             │
│   ┌───────┐     ┌───────┐     ┌───────┐    │
│   │  NVR  │     │미니PC │     │ 인터넷 │    │
│   │       │     │       │     │       │    │
│   └───┬───┘     └───┬───┘     └───┬───┘    │
│       │             │             │        │
│    유선 LAN      유선 LAN       WAN        │
│                                             │
└─────────────────────────────────────────────┘
```

### 필수 조건

- 미니PC ↔ NVR: **같은 네트워크** (192.168.x.x)
- 유선 연결 필수 (Wi-Fi 불안정)
- 인터넷 업로드 속도: 최소 10Mbps (채널당 2-3Mbps)

### IP 설정 예시

| 장치 | IP | 용도 |
|-----|-----|-----|
| 공유기 | 192.168.0.1 | 게이트웨이 |
| NVR/DVR | 192.168.0.100 | RTSP 소스 |
| 미니PC | 192.168.0.10 | 중계 서버 |

## 전력 및 UPS

### 소비 전력 예상

- 미니PC (N100): ~10-15W
- 월 전기료: 약 3,000원

### UPS 추천 (선택)

정전 시 안전 종료를 위해:

- **APC BK650-KR** - 650VA, ~8만원
- 약 20분 백업 가능

## 설치 위치

- NVR 근처 (RTSP 지연 최소화)
- 환기 가능한 곳 (발열 관리)
- 유선 LAN 연결 가능한 곳
